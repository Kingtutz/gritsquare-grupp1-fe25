About Garden Gathering

Garden Gathering is a social, interactive web app where every message becomes a flower in a shared digital garden. Instead of posting in a classic feed, users plant thoughts directly into a playful visual space and can move through a living scene with animated elements.

Users can sign up, log in, write posts, drag flowers around the garden, react to other posts, and explore features like moving clouds, animated animals, and personal user spaces. The app is powered by Firebase Realtime Database for live data syncing and Firebase Auth for secure account handling.

Built by FE25 Group 1, the project combines frontend design, animation, realtime data and teamwork to create a messageboard that feels more like a small online world than a traditional forum.
